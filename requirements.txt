# ============================================================================
# Installation Notes
# ============================================================================
#
# 1. Install all dependencies:
#    pip install -r requirements.txt
#
# 2. First-time TTS setup:
#    - Coqui TTS will download model (~100MB) on first run
#    - This is automatic and happens only once
#
# 3. PyAudio installation on Windows:
#    - If pip install fails, download wheel from:
#      https://www.lfd.uci.edu/~gohlke/pythonlibs/#pyaudio
#    - Install with: pip install PyAudio-0.2.14-cp310-cp310-win_amd64.whl
#
# 4. MongoDB:
#    - Required for saving leads, bookings, and call logs
#    - Start with: docker-compose up -d mongodb
#    - Or install locally from: https://www.mongodb.com/
#
